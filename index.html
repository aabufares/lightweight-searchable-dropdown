Searchable Dropdown Component

A lightweight and customizable dropdown for advanced functionality.

Features

  • Search functionality for filtering options dynamically.
  • Highlighting the selected option.
  • Support for enabling/disabling options with an isActive flag.
  • Dynamic updates to options at runtime.
  • Visual arrow indicators for dropdown state.
  • Fully customizable styles using CSS.

Installation

  1. Clone the repository: 
    git clone https://github.com/aabufares/lightweight-searchable-dropdown.git
  2. Include the dropdown.js and style.css files in your project.

Usage

<div id="dropdown"></div>
<script src="dropdown.js"></script>
<script>
const dropdown = new SearchableDropdown("dropdown", {
    options: [
        { value: "option1", label: "Option 1", isActive: true },
        { value: "option2", label: "Option 2", isActive: false },
        { value: "option3", label: "Option 3", isActive: true }
    ],
    placeholder: "Type to search...",
    required: true,
    onchange: (selectedItem) => console.log("Selected:", selectedItem)
});
</script>

Options Configuration

Customize the behavior and appearance of the dropdown:

  • options: Required array of option objects with value, label, and isActive properties.
  • placeholder: Optional placeholder text when no value is selected. Default: "Search...".
  • defaultValue: Optional preselected value. Must match a valid value from options.
  • required: Optional boolean to mark the field as required. Default: false.
  • disabled: Optional boolean to disable the dropdown. Default: false.
  • onchange: Optional callback triggered on selection. Returns the selected item's value and label.

Examples

Explore various scenarios in the examples folder:

  • basic-example.html: Basic implementation with static options.
  • advanced-example.html: Demonstrates features like inactive options and highlighting.
  • dynamic-update.html: Illustrates dynamic updates.
  • casecade-example.html: Demonstrates cascading dropdown behavior.

1. Basic Example

const dropdown = new SearchableDropdown("basicDropdown", {
    options: [
        { value: "option1", label: "Option 1", isActive: true },
        { value: "option2", label: "Option 2", isActive: true },
        { value: "option3", label: "Option 3", isActive: true }
    ]
});

2. Dynamic Updates

dropdown.updateOptions([
    { value: "newOption1", label: "New Option 1", isActive: true },
    { value: "newOption2", label: "New Option 2", isActive: true },
    { value: "newOption3", label: "New Option 3", isActive: false } // Disabled
]);

3. Advanced Example

const dropdown = new SearchableDropdown("advancedDropdown", {
    options: [
        { value: "advanced1", label: "Advanced Option 1", isActive: true },
        { value: "advanced2", label: "Advanced Option 2", isActive: false },
        { value: "advanced3", label: "Advanced Option 3", isActive: true }
    ],
    placeholder: "Choose an advanced option...",
    defaultValue : "advanced3",
    onchange: (selectedItem) => console.log("You selected:", selectedItem)
});

4. Disabled/Readonly Dropdown

const dropdown = new SearchableDropdown("basicDropdown", {
    options: [
        { value: "option1", label: "Option 1", isActive: true },
        { value: "option2", label: "Option 2", isActive: true },
        { value: "option3", label: "Option 3", isActive: true }
    ],
    disabled: true
});
